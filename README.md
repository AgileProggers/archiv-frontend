### Local dev

[rollup.config.js](rollup.config.js) contains the api endpoint. Dev mode uses my private url, which is not reachable from outside my lan. Just change `ENV_BASE_URL` to `https://archiv.wubbl0rz.tv`.

```
npm i
npm run dev
```
